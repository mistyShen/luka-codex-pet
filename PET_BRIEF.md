# Pet Brief — 巡音流歌 Codex 宠物

## 基本信息

- 宠物名称：巡音流歌 Codex Pet
- `pet_id`：luka-codex
- 一句话描述：一只安静优雅的粉色章鱼歌姬，在屏幕角落陪你写代码、看日志、检查结果。
- 主要用途/感觉：温柔陪伴、冷静提醒、轻成熟助手感。整体感觉应比诗岸更成熟、更从容，像一个低声提醒你“先看日志”的粉色小助手。

## 外观

- 物种/形态：Q 版拟人少女 + 小章鱼元素。主体是 2.5 到 3 头身 Q 版少女，旁边或怀里可以有一只小粉章鱼作为辅助意象。
- 主要颜色：粉色、黑色、金色、白色。头发以柔和粉色为主，服装以黑金为主，局部可加入白色高光。
- 材质/画风：Q 版二次元桌宠风。线条清晰，色块干净，少量柔和高光。整体要可爱但不幼稚，优雅但不复杂。
- 表情：默认平静、温柔、略带成熟感。表情幅度克制，不要大笑大哭；可以有轻微微笑、眨眼、思考、困惑、认真、疲惫等状态。

### 重要特征

- 粉色长发，发尾柔顺，有轻微卷曲感。
- 紫色或蓝紫色眼睛。
- 黑金配色的简化歌姬服装。
- 颈部或胸前可有金色装饰。
- 可加入耳机、麦克风或音符元素，但必须简化，不能有可读 logo 或文字。
- 代表物建议使用小粉章鱼或金色音符麦克风。
- Codex 场景中，小章鱼可以抱着日志本、键盘、终端窗口形小板或进度条形道具；这些道具需要无文字、无 UI 细节，并且贴近角色或被角色拿着。

### 禁止出现的元素

- 过度性感化设计。
- 过分复杂的成人比例。
- 太强的赛博机甲风。
- 大面积霓虹光污染。
- 恐怖章鱼、触手怪、黏液感。
- 过度夸张表情，比如大哭、大喊、大笑。
- 过多文字气泡遮挡角色。
- 与原角色气质严重不符的幼稚吵闹风。
- 可读商标、角色名文字、歌词、UI 截图、水印。
- 漂浮文字、省略号、红叉、代码、音符粒子或进度粒子；如需表达语义，用角色姿态、手持无字道具或小章鱼动作表达。

## 动作偏好

- `idle`：流歌安静坐着或站着，长发轻轻摆动。小章鱼可以趴在她旁边，缓慢眨眼。整体像在听一段很轻的旋律，不主动打扰用户。
- `running-right`：向右小步移动，长发和裙摆轻轻滞后摆动。小章鱼可以跟在后面小跑，或者被她轻轻抱着移动。动作柔和，不像奔跑冲刺。
- `running-left`：与向右移动对称。身体微微向左倾，发尾慢慢跟随。小章鱼保持可爱圆润，不做夸张弹跳。
- `waving`：轻轻抬手挥一下，幅度小，表情淡淡微笑。小章鱼可以举起一只小触手同步挥动。
- `jumping`：轻轻小跳，不要高跳。发尾和衣摆稍微上扬，小章鱼可以被弹起一点点。感觉是任务完成后的轻微开心，不是兴奋乱跳。
- `failed`：流歌低头看着小章鱼或手持无字终端小板，表情微微困惑。小章鱼可以抱着一个小小的红色错误色块或无字标记。整体表达冷静处理错误，不表现崩溃。
- `waiting`：流歌抱着小章鱼坐着，轻轻眨眼。状态像“我在等你确认”，不催促；不要画文字省略号，用姿态和停顿感表达等待。
- `running`：Codex 工作中。流歌看向小型无字终端板或金色音符麦克风，小章鱼可以敲小键盘或抱着日志卷轴。表现正在处理任务，不画浮动代码、进度粒子或普通跑步动作。
- `review`：流歌拿着无字结果板或小型报告卡片，神情认真。小章鱼可以举着一个小勾形道具。整体表达结果已整理，可以检查。
- `complete`：参考图中有完成/成功状态，但 Codex 宠物当前合同只使用 9 行状态，不单独生成 `complete` 行；其轻松成功情绪可并入 `jumping` 或 `review`。

## 备注

- 可以使用“巡音流歌”的角色灵感进行私人桌宠设计参考。如果用于公开发布、商用、分发或品牌化项目，需要注意相关版权与角色使用边界，避免声称官方授权。
- 允许补充生成 Q 版三视图、动作帧、表情表、代表物、小章鱼配件、颜色板和 Codex 工作状态素材。
- 整体保持成熟温柔，不要做成吵闹型宠物。
- Q 版比例要明显，头大身小，适合桌面小尺寸显示。
- 线条存在感要强，缩小后仍然清晰。
- 小章鱼要可爱、圆润、干净，不要有怪物感。
- 动作语义要清楚，但幅度要克制。
- 不要过度还原原角色服装细节，优先保证识别度和桌宠可读性。
- Codex 场景下，她更像温柔的代码审阅搭档，而不是纯装饰角色。

## 生成安全解释

- 最终 spritesheet 要纯透明背景；生成源图使用纯 `#00FF00` chroma-key 后处理。
- 所有 row 必须保持同一角色身份、比例、脸、发型、服装配色、小章鱼或麦克风道具。
- 每个动作 row 内不能跨格、不能裁切、不能出现 guide marks。
- 道具必须贴近角色、被角色拿着或被小章鱼抱着；不要漂浮在角色外侧。
- `running-left` 可以在 `running-right` 视觉稳定后镜像派生；如果耳机、麦克风或小章鱼位置导致方向不自然，则单独生成。
